# Stock Analysis Skill

快速股票分析和财务数据收集技能。

## 功能

1. 使用 searxng 搜索股票基本信息
2. 收集财务数据（营收、利润、市值等）
3. 分析风险因素
4. 生成投资建议和建仓策略

## 使用方法

当用户要求分析某只股票时：

1. 使用 searxng skill 搜索股票代码和基本信息
2. 搜索最新财报数据
3. 搜索分析师评级和目标价
4. 搜索行业竞争和风险因素
5. 综合分析生成报告

## 报告结构

```markdown
# [股票名称] ([股票代码]) 分析报告

## 基本信息
- 当前股价
- 市值
- 市盈率
- 涨跌幅

## 财务状况
- 营收趋势
- 利润情况
- 现金流
- 负债率

## 风险分析
- 行业风险
- 公司特定风险
- 市场风险

## 投资建议
- 综合评级
- 目标价位
- 建仓策略
- 止损位

## 关键观察指标
```

## 工具

- searxng: 信息搜索
- pandas: 数据处理（如果需要）
- 生成 Word/Markdown 报告
