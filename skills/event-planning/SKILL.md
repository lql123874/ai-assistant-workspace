# Event Planning Skill

活动策划和执行方案生成技能。

## 功能

快速生成专业的活动策划文档，包括：
- 活动概述和目标
- 详细执行流程
- 时间线和里程碑
- 团队分工
- 风险管理
- 应急预案

## 文档模板

### 基础策划文档
```markdown
# [活动名称] 策划方案

## 活动概述
- 活动主题
- 活动目标
- 目标受众
- 活动时间和地点

## 活动规则
- 参与条件
- 报名方式
- 评选标准

## 执行流程
1. 前期准备
2. 活动执行
3. 后期跟进

## 预期效果
```

### 详细执行文档
```markdown
# [活动名称] 详细执行流程

## 时间线
| 时间 | 阶段 | 负责人 | 关键任务 |

## 团队分工
| 角色 | 职责 | 人员 |

## 风险管理
| 风险 | 等级 | 应对措施 |

## 应急预案
```

## 输出格式

- Markdown 格式（便于编辑）
- Word 文档（正式分发）
- 可选：海报/宣传图

## 工具

- python-docx: 生成 Word 文档
- Pillow: 生成海报（如果需要）
